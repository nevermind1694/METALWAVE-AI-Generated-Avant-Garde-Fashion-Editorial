# METALWAVE — AI-Generated Avant-Garde Fashion Editorial

## Discription:
METALWAVE is an AI-powered avant-garde fashion editorial created using Shakker AI. The project explores oversized metallic silhouettes, liquid-chrome textures, and futuristic sculptural streetwear for plus-size Gen Z women. By merging generative AI with fashion experimentation, METALWAVE reimagines inclusivity through high-fashion futurism.

## Workflow & Settings
**Tool:** Shakker AI

**Model Used:** Realistic / Photographic Fashion Model

**Aspect Ratio:** 16:9

**Creativity:** Medium–High

**Lighting:** Soft Studio / Rim Light


**Prompt:**
“Avant-garde fashion editorial featuring a plus-size Gen Z female model wearing an oversized sculptural metallic outfit made of chrome, liquid-silver textures, glossy reflective folds, asymmetrical shapes inspired by Issey Miyake x cyberpunk, high-fashion dramatic posing, clean gradient backdrop, soft cinematic rim lighting, photorealistic details, 50mm DSLR lens, ultra-detailed fabric reflections, surreal futuristic streetwear aesthetic.”


**Negative Prompt:**
“distorted hands, extra limbs, melted faces, text, watermark, deformed anatomy, glitch fabric”
